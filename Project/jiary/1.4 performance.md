## meta

> [참고](https://wormwlrm.github.io/2023/05/07/SEO-for-Technical-Blog.html)

meta 데이터는 Next.js의 `Head` 컴포넌트로 페이지 별로 삽입할 수 있다. page 컴포넌트에만 적용해주었다.

- title: 제목
- description : 설명
- author : 작성자
- keywords : 키워드
