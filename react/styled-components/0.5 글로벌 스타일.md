# 글로벌 스타일

styled-components에서 글로벌 스타일을 추가하고 싶을 때는 `createGlobalStyle` 이라는 것을 사용한다.

```jsx
import { createGlobalStyle } from 'styled-components';

const GlobalStyle = createGlobalStyle`
  body {
    background-color: #e9ecef;
  }
`;

function App() {
  return (
    <>
      <GlobalStyle />
      <div>안녕하세요</div>
    </>
  );
}

export default App;
```

이렇게 설정해줄 수 있다.
