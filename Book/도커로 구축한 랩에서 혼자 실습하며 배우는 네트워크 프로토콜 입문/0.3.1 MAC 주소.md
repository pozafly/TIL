# MAC 주소

- LAN 에 연결된 단말의 식별 ID.
- '00-0c-29-43-5e-be'나 '04:0c:ce:da:3a:6c' 와 같이 1 바이트를 하이픈과 콜론으로 구분해 16 진수로 표기.
- **물리 머신**: 물리 NIC(네트워크카드)를 제조할 때 ROM(Read Only Memory)에 기록.
- **가상 머신**: 하이퍼바이저가 가상 NIC에 할당.
	- 컨테이너의 경우, 기본적으로 부팅 시 할당된 IP 주소에서 자동으로 MAC 주소 생성됨.

## 특별한 의미의 비트

![[assets/images/865b0fdffd4428ce0afa50b7b877a340_MD5.png]]

- 8번째 비트인 `I/G 비트` (Individual/Group 비트)
	- 통신의 종류
- 7번째 비트인 `U/L 비트` (Universally/Locally Administered 비트)
	- 운영관리 방식

### I/G 비트

- MAC 주소가 유니캐스트 주소인지, 멀티캐스트 주소인지를 나타냄.
- `0`: 유니캐스트
- `1`: 멀티캐스트 그룹
	- 멀티캐스트 주소는 발신자 MAC 주소로 설정되지 않음.
	- 항상 목적지 MAC 주소로 설정됨.
- 참고로 멀티캐스트 주소 중 48비트가 모두 '1'인 MAC 주소(ff:ff:ff:ff:ff:ff)를 '브로드캐스트 주소'라 하고, 같은 LAN(이더넷 네트워크)에 접속하는 모든 단말을 나타냄.

![[assets/images/95cd6eb14f009cf952fa8a7953df2ac4_MD5.png]]

### U/L 비트

- MAC 주소가 IEEE에서 운영되는 '범용 주소'인지, 조직 내 관리되는 '로컬 주소'인지를 나타내는 값.
- `0`: IEEE에 할당, 전 세계에서 유일무이한 MAC 주소.
- `1`: 관리자가 자체적으로 설정하고나 OS의 기능으로 무작위로 할당된 MAC 주소.
