# NAT

- IP 주소를 변환하는 기술
- 변환 전후의 IP 주소와 포트 번호를 '**NAT 테이블**'이라는 메모리상 테이블에 관리

![[Pasted image 20260119214120.png]]

## 정적 NAT (static NAT)

- 외부와 내부 IP 주소를 **1:1**로 묶어 변환.
- 1:1 NAT라고도 불림.
- 미리 NAT 테이블에 외부 IP주소와 내부 IP 주소를 고유하게 묶는 NAT 항목을 가지고 있음.
- 서버를 인터넷에 공개하거나 특정 단말이 특정 IP 주소로 인터넷에 접속하도록 하고 싶을 때 사용

## NAPT

- Network Address Port Translation
- 내부와 외부의 IP 주소를 **n:1**로 묶어 변환.
- `내부 IP 주소 + 포트번호` 와 `외부 IP 주소 + 포트번호` 를 고유하게 묶는 NAT 항목을 NAT 테이블에 동적으로 추가 및 삭제함.
