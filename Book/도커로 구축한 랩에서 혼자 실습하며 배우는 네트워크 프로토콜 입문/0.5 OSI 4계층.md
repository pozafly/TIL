# OSI 4계층

## OSI 4계층(전송 계층)

- 애플리케이션 식별, 요구사항에 따른 전송 제어 역할
- 네트워크 계층은 여러 네트워크를 넘어 패킷 전달까지가 그 역할. 그이상 no
- ip 덕분에 해외 서버에 접속 가능하지만 패킷을 어떤 애플리케이션에 전달하는지 알 수 없음.

![[Pasted image 20260120000613.png]]

- '**포트 번호**' 라는 숫자를 이용해 패킷을 전달할 애플리케이션을 식별함.
- 패킷의 송수신량을 조절, 전송 도중 사라진 패킷을 재전송하기도 함.

프로토콜
- UDP(User Datagram Protocol): 즉시성(실시간성) 특화
- TCP(Transmission Control Protocol): 안정성 특화
