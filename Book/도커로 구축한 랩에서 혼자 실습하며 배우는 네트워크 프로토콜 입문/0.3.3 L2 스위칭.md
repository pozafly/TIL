# L2 스위칭

- L2 스위치는 이더넷 헤더에 포함된 발신자 MAC 주소와 자신의 포트 번호를 관리.
- 이더넷 프레임의 전달처를 전환하는 기술을 '**L2 스위칭**'
- 발신자 MAC 주소와 포트 번호를 관리하는 표를 '**MAC 주소 테이블**'

## MAC 주소 중복 시 동작

MAC 주소가 최근엔 고유하지 않음. MAC 주소 테이블이 계속 최신의 것으로 덮어써짐.
