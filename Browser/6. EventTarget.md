# EventTarget

모든 Node는 EventTarget을 상속받는다. addEventListener(), removeEventListener(), dispatchEvent() 는 EventTarget에서 지원하는 Method.

- childNodes : 자식 요소를 다 가져옴
- nextSibling : 다음 요소를 가져옴
- parentNode : 부모 요소를 가져옴
