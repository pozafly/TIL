# 라이프사이클 메서드

```html
<script>
  export default {
  // 여기에 작성한다.
  }
</script>
```

## 대표적인 라이프사이클 메서드

1. `created` : 인스턴스가 생성되자마자 호출되는 hook.

2. `mounted`  : DOM이 마운트 된 후

3. `updated`  : data가 업데이트 된 후

4. `destroyed` : vue 인스턴스가 사라진 후

beforeCreate, beforeMount, beforeUpdate, beforeDestory 등등 있다. 그 전에 실행되는 hooks.
