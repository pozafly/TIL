# 이미지 비율

- 멀티미디어 요소들도 가변적으로 작동하게 만들어 주어야 함.
- 이미지와 동영상 요소를 너빗값과 최대 너빗값을 100%로 설정하면 멀티미디어 요소가 가변적으로 동작함

```css
img, video {
  width: 100%;
  max-width: 100%;
}
```

<br/>

## Width, max-width 차이점

- width: 속성값을 100%으로 둘 경우 요소 크기를 100%로 유지
- max-width: 100%로 설정할 경우 요소의 기본 크기 이상으로는 크기가 조절되지 않는다.
